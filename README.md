# Currency Converter

A responsive Currency Converter web application built using HTML, CSS, and JavaScript. The application fetches real-time exchange rates and allows users to convert currencies instantly with dynamic country flag updates.

## Features

* Real-time currency conversion
* Supports multiple international currencies
* Dynamic country flags based on selected currencies
* Responsive and user-friendly interface
* Live exchange rate data

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Currency API
* Flags API
* Font Awesome

## How to Run

1. Clone the repository

```bash
git clone https://github.com/prince444thakur/currency-converter.git
```

2. Open the project folder

```bash
cd currency-converter
```

3. Open `index.html` in your browser
